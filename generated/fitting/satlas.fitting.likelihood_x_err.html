satlas.fitting.likelihood_x_err¶

satlas.fitting.likelihood_x_err(spectrum, x, y, s, func)[source]¶

Calculates the loglikelihood for a spectrum given x and y values. Incorporates a common given error on the x-axis.

Parameters:
  • spectrum (Spectrum) – Spectrum object set to current parameters.
  • x (array_like) – Experimental data for the x-axis.
  • y (array_like) – Experimental data for the y-axis.
  • s (float) – Experimental uncertainty on the x-axis
  • func (function) – Function taking (y_data, y_model) as input, and returning the loglikelihood that the data is drawn from a distribution characterized by the model.
Returns:
Return type:

array_like

Note

This method uses the FFT algorithm to quickly calculate a convolution integral. If greater accuracy is required, change satlas.fitting.theta_array to a suitable range and length.
